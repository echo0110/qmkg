# 全民k歌导出工具

全民导歌 `V1.2`  [下载](https://github.com/zaxtyson/qmdg/raw/master/files/全民导歌_1.2.apk)

* 修复了手贱多改了一个字符导致的应用奔溃问题

> 食用方法:打开全民k歌，点开音乐播放界面，点右上角分享，选择复制链接，然后打开本软件，点解析就好了。复制全民k歌网页版的链接也是可以的。
